# Code of Conduct

## Purpose

Our project is committed to providing a welcoming and inclusive environment for all contributors, regardless of their background, experience, or identity. This Code of Conduct outlines our expectations for behavior and the consequences for violating these expectations.

## Expected Behavior

We expect all contributors to:

- Be respectful and considerate of others.
- Communicate openly and constructively.
- Be inclusive and welcoming to new contributors.
- Respect differences of opinion and experience.
- Follow the project's guidelines and processes.

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or hate speech.
- Personal attacks or insults.
- Disruptive or disrespectful behavior.
- Unwanted sexual attention or advances.
- Publishing others' private information without permission.
- Any behavior that violates applicable laws or regulations.

## Consequences

Violations of this Code of Conduct may result in:

- A warning from the project maintainers.
- Temporary or permanent removal from the project.
- Reporting to appropriate authorities in cases of illegal behavior.

## Reporting Guidelines

If you witness or experience behavior that violates this Code of Conduct, please report it to the project maintainers. You can do this by:

- Opening an issue on our GitHub repository.
- Sending an email to fiszki[at]quickclicker.click.
- Contacting a project maintainer directly.

All reports will be handled confidentially and investigated promptly.

## Scope

This Code of Conduct applies to all project-related spaces, including but not limited to:

- GitHub repository and issues.
- Project forums or mailing lists.
- Social media channels.
- In-person events or meetings.

## Enforcement

The project maintainers are responsible for enforcing this Code of Conduct. We will investigate all reports and take appropriate action based on the severity of the violation.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.0, available at [https://www.contributor-covenant.org/version/2/0/code_of_conduct.html](https://www.contributor-covenant.org/version/2/0/code_of_conduct.html).

## Community Impact Guidelines

1. **Correction**: For minor violations, the maintainers may issue a warning and request a correction.
2. **Warning**: For more serious violations, the maintainers may issue a formal warning.
3. **Temporary Ban**: For repeated or severe violations, the maintainers may impose a temporary ban from the project.
4. **Permanent Ban**: For extreme violations or repeated offenses, the maintainers may impose a permanent ban from the project.

Thank you for helping to make our project a welcoming and inclusive space for everyone.
